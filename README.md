# 🎓 Terceiro Semestre — Análise e Desenvolvimento de Sistemas (ADS)

Este repositório reúne os materiais desenvolvidos durante o **3º semestre do curso de Análise e Desenvolvimento de Sistemas (ADS)**.

O objetivo é documentar minha evolução acadêmica e técnica, registrando projetos, atividades práticas, experimentações e conteúdos estudados ao longo do semestre, com foco em desenvolvimento de software, computação em nuvem, integração industrial e tecnologias da Indústria 4.0.

---

## 📚 Disciplinas e Projetos

### 🤖 Chatbot

Desenvolvimento de aplicações conversacionais utilizando conceitos de automação, integração de serviços e interação inteligente com usuários,  utilizando Langflow.

### ☁️ Cloud Computing

Estudos e práticas relacionadas à computação em nuvem com a plataforma Azure, infraestrutura virtualizada, deploy de aplicações e utilização de máquinas virtuais.

### 📱 Desenvolvimento Mobile

Criação de aplicações mobile utilizando Dart com o framework Flutter, navegação e estruturação de aplicativos multiplataforma.

### 🏭 IIOT (Industrial Internet of Things)

Aplicação de conceitos de Internet das Coisas voltados ao ambiente industrial, integração entre dispositivos e coleta de dados em sistemas conectados.

### 🔗 Integração Industrial

Fundamentos de comunicação entre sistemas industriais, integração de processos e utilização de tecnologias voltadas à automação industrial.

### ⚙️ Indústria 4.0

Estudo das tecnologias emergentes da transformação digital industrial, incluindo automação inteligente, conectividade e sistemas ciberfísicos.

### 🚀 Projeto de Extensão

Desenvolvimento de um aplicativo mobile institucional para o recanto dos velhinhos de Valinhos.
